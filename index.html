Extension is a mod for GWater2 with features like:

- Drowning
- More Parameters related to:
   - Drowning
   - Visuals
   - Splash Sound
   - Deadly Liquid
- Custom Preset Support with "--" comment support
- Extension Parameter Preset Support
- Stats in the left upper corner (inpired by gwater)
